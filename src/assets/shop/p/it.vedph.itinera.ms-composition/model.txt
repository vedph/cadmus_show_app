- sheetCount* (int)
- guardSheetCount (int)
- guardSheets (MsGuardSheet[]):
  - isBack (boolean)
  - material (string, thesaurus ms-materials)
  - location (MsLocation):
    - n* (int): sheet number.
    - r (boolean): number is Roman.
    - s (string): suffix, containing 0-2 lowercase letters (e.g. r=recto, v=verso, rv=both).
    - l (int): line number.
  - date (HistoricalDate)
  - note (string)
- sections (MsSection[]):
  - tag (string): any arbitrarily chosen ID used to group sections together.
  - label* (string)
  - start* (MsLocation)
  - end* (MsLocation)
  - date (HistoricalDate)
